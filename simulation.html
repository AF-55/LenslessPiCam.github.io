Simulations

We subjected each mask variant to the same simulation and reconstruction in order to determine which one is the most performant. Since all masks are not separable, the lensless capture simulation has been done using the convolution model from waveprop: $$ Y = A * X $$ The reconstruction has been done with the ADMM algorithm (Alternating Direction Method of Multipliers).
